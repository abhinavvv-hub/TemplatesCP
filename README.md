# 📚 TemplatesCP

A curated collection of competitive programming templates that I use during contests and practice. This repository contains clean, optimized, and reusable implementations of commonly used algorithms and data structures, allowing faster problem solving without rewriting boilerplate code.

## ✨ Features

* 🚀 Contest-ready C++ templates
* ⚡ Optimized implementations with competitive programming constraints in mind
* 🧩 Reusable snippets for common algorithms and data structures
* 📖 Well-organized code for quick reference during practice
* 🔄 Continuously updated as I learn new techniques

## 📂 Contents

The repository includes templates for topics such as:

* Graph Algorithms
* Trees & Binary Lifting
* Lowest Common Ancestor (LCA)
* Shortest Paths
* Disjoint Set Union (DSU)
* Segment Trees
* Fenwick Tree (BIT)
* Sparse Table
* Dynamic Programming
* Number Theory
* Modular Arithmetic
* String Algorithms
* Trie
* Rolling Hash
* Prefix Sum & Difference Array
* Sliding Window & Two Pointers
* Binary Search
* Greedy
* Geometry
* Miscellaneous Utilities

> More templates are added regularly.

## 🛠️ Usage

Clone the repository:

```bash
git clone https://github.com/abhinavvv-hub/TemplatesCP.git
```

Copy the required template into your solution or include it in your personal competitive programming setup.
